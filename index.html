Day Month Year:

Parameters of the above:

format "dd/MM/yyyy" provide the date format.
start_time "05/11/2450" provide a datestring for initiation.
range false no splitting button, given only one input.
labels true show labels over wrap input field.
aware_label true names of days and months. requires labels parameter set to true.
labels_days ["Κυρ", "Σαβ", "Παρ", "Πεμ", "Τετ", "Τρι", "Δευ"] Example array of day names in greek, default is english.
labels_mths ["Jek", "Qañ", "Aqp", "Naw", "Säw", "Mam", "Maw", "Şil", "Tam", "Qır", "Qaz", "Qar" ] Example array of month names in kazakh, default is english.
ast_value true assert date value.
upstream_input false prevent upstream input.
keyboard_arrow false prevent keyboard arrow input.
